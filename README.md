# spider_multiprocess_jinritoutiao_images

1.爬取对象

今日头条（https://www.toutiao.com/），是一个综合性信息平台；

今日头条图片站（https://www.toutiao.com/ch/news_image/）提供海量的图片内容，用户可以通过输入框搜索的形式获得自己感兴趣的内容。

本文介绍的方法，用于爬取今日头条图片站‘街角’内容。
